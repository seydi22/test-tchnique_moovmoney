4. Intégration de Moov Money dans un site e-commerce

Voici comment connecter une boutique en ligne à la plateforme de paiement Moov Money.

1. Demande d'accès API

Contacter Moov pour obtenir les identifiants API (Client ID, Secret, clé de signature, URL des endpoints).

2. Création du compte marchand

Créer un compte commercial Moov pour recevoir les paiements clients.

3. Intégration technique

Depuis le site e-commerce, envoyer une requête POST vers l’API Moov contenant les infos de paiement.


POST /api/moov/payment
{
  "amount": 5000,
  "phone": "48503366",
  "reference": "CMD_20250622"
}

4. Callback & confirmation

Moov envoie une réponse (callback) à ton serveur pour indiquer si le paiement a réussi. Tu mets alors la commande à jour.

5. Sécurité

Signer les requêtes, utiliser HTTPS, vérifier les signatures de Moov et stocker les logs de transaction.
